Zush runs on both Mac (macOS 15+) and Windows (10/11) and offers two PRO plans: $8/month or $38 one-time, both with unlimited PRO renames. BYOK remains available when you want to use your own AI provider key. Version 3.0 supports 100+ formats: 41 image formats including RAW, HEIC, AVIF, SVG, and TIFF; 4 design formats including Sketch, Figma .fig, Adobe Illustrator .ai, and PSD; 37 document and PDF-related formats including Office, iWork, OpenDocument, text, CSV/TSV, JSON, XML/YAML, EML, SRT, and VTT; 16 video formats including MP4, MOV, M4V, MPEG, 3GP, TS, MTS, M2TS, DV, and VOB; and 7 audio formats including MP3, M4A, WAV, FLAC, OGG, WebM, and MPGA.

For Mac users, there is also Offline AI mode via Ollama where files never leave the device.

NameQuick is a Mac-only AI file renamer whose current public homepage lists a Managed path at $12/mo or $12/year and a Self-Managed plan at $69 one-time. The Self-Managed plan is the BYOK/local-model path: you bring your own cloud API key or run a local model on your Mac. The old $5/$10/$20/$35 monthly tier table is no longer the safe comparison copy.

Their AI provider menu is broader than most: OpenAI, OpenRouter, Claude, Gemini, Ollama, and LM Studio — six public options including two local options. The privacy story is strong: "EU-routed & encrypted" with "no permanent storage" and "data is never used to train AI" per their site.

The Rules Engine is one of NameQuick's distinct features — beyond renaming, it can move, tag, archive, and reorganize files automatically based on conditions you define.

## Format support

### RAW photos: Zush yes, NameQuick unclear

Zush publishes a complete RAW format list (CR2, CR3, NEF, ARW, DNG, ORF, RAF, RW2, PEF, SRW, SR2). NameQuick lists "30+ file formats" on their site but does not enumerate camera RAW formats specifically. If you shoot RAW, Zush is the safer pick.

### Office docs and OCR: both cover the common case

NameQuick advertises OCR for scanned docs and EXIF extraction for photos. Zush supports PDF, DOCX, XLSX, PPT, PPTX, TXT, MD, JSON, EML, CSV, DOC, plus iWork files such as Pages, Numbers, and Keynote. Both handle the everyday office formats.

### Audio / video / archives

NameQuick mentions audio, video, and archives as part of its 30+ formats. Zush 3.0 supports common video files, subtitle sidecars, design files, iWork documents, and audio files including MP3, M4A, WAV, FLAC, OGG, WebM, and MPGA. NameQuick is still broader if ZIP/archive renaming is central to your workflow.

## AI provider menu

NameQuick publicly lists six AI backends: OpenAI, OpenRouter, Claude, Gemini, Ollama, and LM Studio. Zush supports 4 cloud providers: Gemini, Groq, OpenAI, and Claude — plus Ollama for the Offline AI mode.

The big differentiators:

- <BlogHighlight>Groq</BlogHighlight> — listed on Zush, not on NameQuick's public provider list.
- <BlogHighlight>OpenRouter</BlogHighlight> — only on NameQuick. OpenRouter aggregates 50+ models, useful if you want to test alternatives.

## Pick by use case

**Pick Zush if:**

- You need a Windows app, not just Mac
- You want one desktop workflow across Mac and Windows
- You shoot RAW photos and want explicit format support
- You rename design assets, iWork files, audio, videos, and documents in the same folders

## Migration tip

If you are coming from NameQuick to Zush: install Zush, run the 5-file demo on a sample folder, then choose Monthly or One-Time PRO if it fits. If you were on the BYOK plan, you already have an API key — paste it into Zush's BYOK settings and continue with no API change.

If you are coming from Zush to NameQuick (rare, given pricing): if you need OpenRouter or LM Studio specifically, NameQuick adds those. Otherwise the migration mostly just costs more.

## FAQ

### Does NameQuick run on Windows?

No. NameQuick is macOS only. If you need a Windows app, Zush has one in the Microsoft Store.

### Is BYOK free in both tools?

In Zush, BYOK is included with paid PRO plans — no separate BYOK purchase. In NameQuick, the public Self-Managed plan is $69 one-time, after which you bring your own API key or local model.

### Does Zush support all NameQuick AI providers?

No. Zush supports 4 cloud providers (Gemini, Groq, OpenAI, Claude) plus Ollama for the Offline AI mode. NameQuick publicly lists OpenAI, OpenRouter, Claude, Gemini, Ollama, and LM Studio. If specific providers matter, especially OpenRouter or LM Studio, NameQuick is broader.

### Does NameQuick support RAW photos?

NameQuick's public format list says "30+ formats" without enumerating RAW camera variants. Zush explicitly lists 11 RAW formats. Photographers who rely on RAW are safer with Zush.

### Which has better Mac integration?

Both tools integrate with Finder tags. NameQuick adds Finder Comments, the Rules Engine, and a global hotkey. Zush has Finder tags + Spotlight metadata + Offline AI via Ollama. NameQuick has the deeper macOS-native polish; Zush trades some Mac depth for cross-platform support.

## The main tool categories

### Finder and built-in rename tools

Good for replacing text, adding prefixes, and numbering files. Not AI, and not useful when every file needs a unique descriptive name.

### Metadata-driven renamers

Useful when you want EXIF data, dates, or technical fields in the filename. Less useful when the real problem is that the file name says nothing about the image content.

### AI-powered rename tools

These tools inspect the actual file and generate descriptive names from that analysis. This is the category that matters for queries like `ai renamer mac`, `ai image renamer mac`, and `auto image renamer mac`.

Zush is built for mixed folders. You can batch rename files from ordinary Windows and Mac folders without separating screenshots, PDFs, photos, design files, video, and audio into different tools.

## Images and screenshots

Supported visual formats include PNG, JPG, JPEG, WEBP, AVIF, GIF, BMP, TIFF, HEIC, HEIF, SVG, PDF previews, JP2, EXR, TGA, ICO, ICNS, PBM, PGM, PPM, PFM, and common RAW formats such as CR2, CR3, NEF, ARW, DNG, ORF, RAF, RW2, PEF, SRW, SR2, and RAW.

## Documents and PDFs

Zush supports PDFs, plain text, Markdown, JSON, EML, CSV, TSV, LOG, XML, YAML, YML, SRT, VTT, RTF, ODT, ODS, ODP, Pages, Numbers, Keynote, Word, PowerPoint, and Excel formats.

Common Office formats include DOC, DOCX, DOCM, DOTX, DOTM, PPT, PPTX, PPTM, POTX, POTM, PPSX, PPSM, XLS, XLSX, XLSM, XLTX, and XLTM.

## Design files

Supported design formats include Sketch, Figma, Illustrator, and PSD files where previews or metadata are available for analysis.

## Video and audio

Supported video formats include MP4, MOV, QT, M4V, MPEG, MPG, M2V, 3GP, 3GPP, 3G2, 3GP2, DV, TS, MTS, M2TS, and VOB.

Supported audio formats include MP3, M4A, WAV, FLAC, OGG, WEBM, and MPGA. Audio can use metadata, recognition, and transcript context when available.

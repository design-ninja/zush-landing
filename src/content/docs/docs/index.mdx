## What Zush can do

- Batch rename files with AI-generated names based on file content.
- Monitor folders and rename new supported files as they arrive.
- Save repeatable workflows with Templates.
- Build consistent filenames from 145+ Naming Blocks.
- Add custom rename and tagging prompts.
- Generate searchable filenames, tags, and metadata.
- Process images, screenshots, RAW photos, PDFs, Office and iWork documents, design files, subtitles, video, and audio.
- Use managed Cloud AI, BYOK, or Offline AI through Ollama.
- Review every result before applying changes and revert from rename history.

<Aside type="note">
Zush is a naming workflow. It does not replace Finder, File Explorer, Spotlight, Windows Search, or a full digital asset management system. It makes those tools more useful by turning vague filenames into readable, searchable filenames.
</Aside>
